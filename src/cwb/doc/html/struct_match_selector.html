Detailed Description

The MatchSelector structure defines which part of a (standard) query match is returned.

The structure consists of labels defining the start and end of the part to be extracted (or NULL for match/matchend) and optional offsets to be applied.

It is used by the new "show a .. b" syntax introduced into standard queries by CQP v3.4.32, and does not apply to MU or TAB queries.
