Each possible value of the enumeration represents a particular "type" of command that may potentially yield a (sub)corpus.

Enumerator

Activation 

A corpus-activation command.

SetOperation 

A set operation such as intersect or diff.

Assignment

int cqp_parse_file ( FILE *  src,
int  exit_on_parse_errors

Parses a stream for CQP query syntax.

We're assuming that this is a normal pipe, IE, no interactive shenanigans need to be accounted for (either Readline is unavailable, or the cqp program was started without -e.).

Note that cqp_parse_file() fclose()s fd unless it is STDOUT.

See also
cqp_parse_string
Parameters
srcFile handle of the file to parse. The file may have been opened with cl_open_stream() (but also plain FILE * objects are fine too).
exit_on_parse_errorsBoolean: should CQP stop processing this stream on parse errors?
Returns
Boolean: true = all ok, false = a problem.
