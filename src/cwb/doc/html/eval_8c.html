Assigns the range data from a matchlist object (from running a query) to a CorpusList object that will represent the result as a NQR/subcorpus.

This includes the keyword/target data if present.

Parameters
cp(should be cl) The CorpusList object for the query result we are creating. In subquery filter mode (see below) it needs to contain the range data for the activated NQR being filtered.
matchlistMatchlist object array containing the actual query matches.
nr_listsHas no effect on anything, but it will cause an error if it's greater than 1.
keep_old_rangesBoolean. A somewhat opaquely named parameter which switches on/off "subquery filter mode".
