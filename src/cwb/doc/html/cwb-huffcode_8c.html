◆ decode_check_huff()

static void decode_check_huff ( Attribute *  attr,
char *  fname 
)
static

Checks a huffcoded attribute for errors by decompressing it.

This function assumes that compute_code_lengths() has been called beforehand and made sure that the uncompressed token sequence is used by CL access functions.

It exits on error, so there's no return value.

Parameters
attrThe attribute to check.
fnameBase filename to use for the three compressed-attribute files. Can be NULL, in which case the filenames in the attribute are used.
